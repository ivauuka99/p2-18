Наименование: NotePad (Текстовый редактор/блокнот). Версия программы 15.
Применения: Редактирование и изменение текста.
Программное обеспечение: Windows 7/8/10, не требует стороннего ПО.

Программа написана на языке C++ и скомпилирована в C++ Builder 6. Предназначена для редактирование и изменение текста.

Пользователь может редактировать и измененять текст, цифры, шрифт.
В состав программы входят следующие модули:
– пользовательский интерфейс;
– модуль открытия/закрытия файлов и работа с ними;
– модуль интерфейса;

Интерфейсный модуль построен на основном принципе (меню), и поле для текста и статус бара.
Программа эксплуатируется на персональном компьютере (ПК) типа IBM PC/AT. Для работы в диалоговом режиме используется экран дисплея, клавиатура и манипулятор типа "мышь".

Загрузка программы осуществляется путём запуска (exe)файла, который запускает программу.
